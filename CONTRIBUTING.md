# Contributing

Thank you for considering a contribution to the Django Basic Auth Project Generator.

## Project goal

Keep the generator beginner-friendly and close to the simple Django workflow taught in introductory classes. Avoid adding advanced architecture unless it is optional and clearly explained.

## Before submitting a change

1. Create a new branch from `main`.
2. Test the script on Windows PowerShell.
3. Generate a fresh Django project using sample names such as `demoProject`, `demoApp`, and `Env`.
4. Confirm that migrations complete and the registration, login, logout, home, and admin pages work.
5. Keep the creator notices, `LICENSE`, `NOTICE`, and `CITATION.cff` intact.

## Pull requests

Explain:

- What was changed
- Why the change is useful for beginners
- How the change was tested
- Any new command or prompt introduced

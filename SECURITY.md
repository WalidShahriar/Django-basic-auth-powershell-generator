# Security Policy

## Supported version

Security fixes are currently applied to the latest version on the `main` branch.

## Reporting a security issue

Do not publish passwords, tokens, private repository URLs, or personal information in a public issue.

Use GitHub's private vulnerability reporting feature when it is enabled. If it is unavailable, open a short public issue stating that you need a private contact channel, without including the vulnerability details.

This generator is intended for local learning and development. The generated Django development server and simplified authentication views are not intended for direct production deployment.

# Django Basic Auth Project Generator v1.0.0

Initial public release by MD WALID SHAHRIAR ([@WalidShahriar](https://github.com/WalidShahriar)).

## Highlights

- Beginner-friendly interactive PowerShell generator
- Custom virtual-environment name
- Basic Django authentication structure
- Optional email for superuser creation
- VS Code launch without an extra Command Prompt window
- Server launch in Windows Command Prompt
- Apache 2.0 licensing and GitHub citation metadata
